# Prediction New York Taxi Trip Duration

The task was to predict the duration of a cab ride in New York City based on tabular data such as date and time, coordinates of the call point and destination point, number of passengers, etc. 

The data comes from this competition: https://www.kaggle.com/competitions/nyc-taxi-trip-duration

The work involves Exploratory Data Analysis, Feature Engineering, and the usage of linear models.
